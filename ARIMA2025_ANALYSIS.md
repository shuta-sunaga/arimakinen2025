# 有馬記念2025 予想分析

## レース概要
- **日時**: 2025年12月28日（日）15:40発走
- **競馬場**: 中山競馬場 芝2500m（右回り・内回り）
- **格**: GI（第70回）

---

## 中山芝2500mコース特徴

| 項目 | 特徴 |
|------|------|
| **回り** | 右回り（内回り） |
| **コーナー数** | 6回 |
| **直線距離** | 310m（短い） |
| **高低差** | 5.3m（JRA最大） |
| **ゴール前急坂** | 2.2m（2度越え） |

### 求められる適性
- **スタミナ・持久力**: 2度の坂越えでタフな展開
- **パワー**: 急坂で失速しない脚力
- **先行力**: 4角2桁順位からの勝利は2013年以降なし
- **中山適性**: 小回り・起伏への対応力

### 有利な条件
- 好位〜中団で運べる馬
- 5枠の複勝率が最も高い（30.5%）
- 欧州血統（ドイツ牝系など）

## 主要馬の詳細分析

### レガレイラ（3枠5番）⭐ 本命候補

| 項目 | 詳細 |
|------|------|
| **戦績** | 10戦4勝（GI 3勝） |
| **獲得賞金** | 7億215万円 |
| **血統** | 父スワーヴリチャード × 母父ハービンジャー |
| **騎手** | C.ルメール |

**GI勝利**
- 2023年 ホープフルS（中山2000m・右）
- 2024年 有馬記念（中山2500m・右）
- 2025年 エリザベス女王杯、オールカマー

**強み**
- 昨年の有馬記念覇者（中山実績◎）
- 2025年GI 2勝と絶好調
- ファン投票1位（史上最多61万票）
- 牝馬による有馬記念連覇なら史上初

**懸念点**
- 連覇のプレッシャー
- 年末の消耗

---

### ダノンデサイル（5枠9番）⭐ 対抗候補

| 項目 | 詳細 |
|------|------|
| **戦績** | GI 2勝 |
| **購入価格** | 1億4,850万円（2022年セレクトセール） |
| **血統** | 父エピファネイア |
| **騎手** | 戸崎圭太 |

**主な勝利**
- 2024年 日本ダービー（東京2400m・左）
- 2025年 AJCC（中山2200m・右）
- 2025年 ドバイシーマクラシック（海外GI）

**強み**
- 2025年は3戦2勝と充実
- 中山AJCC勝利で右回り適性証明
- 好位追走からの差し脚
- **5枠は有馬記念で最も好成績**

**懸念点**
- 有馬記念は初出走
- 2500mへの距離延長

---

### メイショウタバル（3枠6番）⭐ 穴候補

| 項目 | 詳細 |
|------|------|
| **戦績** | 11戦5勝（GI 1勝、重賞3勝） |
| **獲得賞金** | 4億4,164万円 |
| **血統** | 父ゴールドシップ |
| **騎手** | 武豊 |

**主な勝利**
- 2025年 宝塚記念（阪神2200m・右）逃げ切り
- 2024年 神戸新聞杯、毎日杯

**強み**
- 逃げ馬で展開を支配できる
- 父ゴールドシップとの父子宝塚記念制覇
- 武豊騎手の経験値
- 右回り適性◎

**懸念点**
- 気性難（皐月賞17着、菊花賞16着）
- 逃げられないと脆い
- 有馬記念の厳しいペースへの対応

## 予想：推奨3頭

### 🥇 本命：レガレイラ（5番）
**理由**
- 中山実績が抜群（ホープフルS、有馬記念優勝）
- 2025年はGI 2勝と最も充実
- ルメール騎手との相性◎
- 史上初の牝馬連覇へ視界良好

### 🥈 対抗：ダノンデサイル（9番）
**理由**
- 2025年充実（AJCC、ドバイシーマC優勝）
- 中山AJCC勝利で右回り・坂適性証明
- **5枠**は有馬記念で最好成績の枠
- 好位差しの脚質がコース向き

### 🥉 単穴：メイショウタバル（6番）
**理由**
- 逃げ馬で展開利を得られる可能性
- 宝塚記念の逃げ切り勝ちは本物
- 武豊騎手の有馬記念実績
- 人気落ちで妙味あり
