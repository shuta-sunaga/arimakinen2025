## 5-4. 最終結論

### 本命：レガレイラ（5番）

**選定理由**
1. 昨年覇者で中山2500m実績が唯一無二
2. 秋2戦連勝（オールカマーGII、エリザベス女王杯GI）
3. ルメール騎手×木村厩舎×中山2500m勝率100%
4. 過去データで1番人気が50%の勝率
5. 牝馬初の連覇という歴史的偉業に挑む

**懸念を払拭した理由**
- 宝塚記念11着の不振は、秋2連勝で完全に払拭
- エリザベス女王杯のレコード勝ちは本物の充実ぶり

### 対抗：ダノンデサイル（9番）

**選定理由**
1. 5枠9番は過去データで連対率30.0%（馬番別トップ）
2. 中山AJCC勝利で右回り・坂適性証明
3. ドバイシーマC優勝でGI級の能力証明
4. 好位差しの脚質がコース向き

### 単穴：メイショウタバル（6番）

**選定理由**
1. 宝塚記念の逃げ切り勝ちは本物
2. 武豊騎手が歴代単独トップの5勝目を狙う
3. 人気落ちで妙味あり
4. 逃げ馬として展開を支配できる可能性
